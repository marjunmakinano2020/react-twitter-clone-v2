## Overview

## Live Demo

## Setup

1. Clone this repository

```
	git clone https://github.com/marjunmakinano2020/react-twitter-clone-v2.git
```

2. Install packages

```
	cd react-twitter-clone-v2
	yarn install
```

3. Run

```
	yarn start
```
